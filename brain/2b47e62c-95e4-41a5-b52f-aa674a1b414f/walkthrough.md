## Design & Interactivity Features
1. **Interactive Image Gallery:** Clicking on any gallery item launches an immersive lightbox modal with image navigation controls (Previous, Next, Close) and keyboard support (Left/Right arrows, Escape key).
2. **Auto-Sliding Testimonials:** Patient reviews slide automatically every 5 seconds, featuring smooth transitions and pagination indicator dots.
3. **Animated Contact Form:** Validation is handled dynamically, and submitting the form simulates a mock API request before displaying an elegant "Message Sent!" toast notification.
4. **Google Maps Integration:** A real-time Google Maps iframe is embedded in the contact section to guide patients to the clinic.
5. **Glassmorphism Navbar:** Stays fixed to the top of the viewport with responsive scroll highlights and a custom mobile hamburger menu.

## How to Edit and Preview the Website
1. **To Preview:** Double-click the [index.html](file:///C:/Users/chittoor/.gemini/antigravity/scratch/denta-home-360/index.html) file to open the website directly in any web browser. No local server or build tools are required!
2. **To Edit Text/Photos:** Open [config.js](file:///C:/Users/chittoor/.gemini/antigravity/scratch/denta-home-360/config.js) in any text editor. Modify the strings (such as phone numbers, services, or photo file paths), save, and refresh your browser.
